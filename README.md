# uw-job-find-browser-plugin


## Project Structure
```
upwork-extension/
├── manifest.json
├── popup.html
├── popup.js
├── styles.css
└── icons/ (optional - we'll use emoji as fallback)
```
## Installation Instructions

1. Create a new folder on your computer named `uw-job-find-browser-plugin`
2. Copy all the code files below into the folder with the exact names specified
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" (toggle in top right)
5. Click "Load unpacked" and select the `uw-job-find-browser-plugin` folder
6. The extension icon will appear in your browser toolbar
